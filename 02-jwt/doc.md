### What is json web token (jwt)?
- jwt là token
- dùng để authentication cho ứng dụng thông qua giao thức http.

### Why?
- tăng cường security.
- tiện ích hơn so với password truyền thống.
- xác thực quyền chủ sở hửu.

- có được payload
- chống giả mạo người dùng
